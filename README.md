Synchronized lyrics with the music in the browser.

# How to use
1. Add a song in html/song.mp3
2. Add lyrics to html/lyrics.json (as a array, not dictionary)
3. Start a server
```bash
python3 -m http.server 8080 -d html
# only 'python' for Windows
```
